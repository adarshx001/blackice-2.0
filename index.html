Educational Note

Password Security Principles

A strong password resists brute-force attacks (guessing all combinations) and dictionary attacks (trying common words).

  • Length: The most crucial factor. Each added character exponentially increases time to crack.
  • Complexity: Mixing character types prevents attackers from narrowing down the search space.
  • Unpredictability: Avoid common substitutions (e.g., 'P@ssw0rd') which are already in attacker dictionaries.

Educational Note

Hashing vs Encryption

SHA-256 is a one-way hashing function. It produces a fixed 64-character signature resistant to reversal. Changing one letter completely changes the hash.

Caesar Cipher is a classic, two-way encryption technique. It shifts each letter of the plaintext by a fixed number of positions down the alphabet (e.g., A becomes D if the shift is 3).

  • Warning: Caesar Cipher provides zero real-world security today as it is trivially broken by frequency analysis or brute force (checking all 25 shifts).

Educational Note

Phishing Detection

Phishing is a social engineering attack where bad actors impersonate legitimate entities to steal credentials or data.

Attackers often use deceptive URLs. Common red flags include:

  • IP Addresses: Real companies use domain names. URLs like `http://192.168.1.1/login` are highly suspicious.
  • Typosquatting: Subtle misspellings (e.g., `google.com` vs `go0gle.com`).
  • Excessive Subdomains: `paypal.com-secure.login.verify.domain.com` attempts to hide the real domain (`domain.com`).
  • Tricky Keywords: Urgent words like "verify", "secure", "update", or "login" embedded in strange paths.

Educational Note

Static File Analysis

File extensions alone don't determine if a file is safe. Attackers use multiple techniques to disguise malware:

  • Executables: Files like `.exe`, `.bat`, `.cmd`, `.ps1`, or `.vbs` are inherently risky as they can execute arbitrary code.
  • Double Extensions: Attackers often use names like `invoice.pdf.exe`. Windows might hide the `.exe`, tricking users into thinking it's a PDF.
  • Office Macros: Documents like `.docm` or `.xlsm` contain embedded scripts that can download malware.
  • Archives: `.zip`, `.rar`, or `.iso` files are frequently used to smuggle payloads past email filters.
